# Processing scripts for SFY data

Install using e.g.:

```
$ pip install -e .
```

## Usage

Set environment variables for server and read token:

```
export SFY_SERVER='http://wavebug.met.no:3000'
export SFY_READ_TOKEN='secret'
export SFY_DATA_CACHE='/path/to/data/cache'
```

try out with:

```
sfydata list
```
