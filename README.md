# The small friendly buoy

* [sfy-buoy](sfy-buoy/) - the firmware for the buoy.
* [sfy-data](sfy-data/) - the server scraping or receiving data from deployed
    buoys.
* [sfy-proto](sfy-proto/) - the shared protocol between buoy and server.
